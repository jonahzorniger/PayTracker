# PayTracker

PayTracker is a web-based application used to track Jobs, Pay, and Customers to confirm your being payed correctly from your company. 

Tables:

1. Jobs - Each job or appointment you go to, will have a customer and work type selection
2. Work Types - Create a type of work type (examples: Quote, Installation, Service, Maintanence.)
3. Customers - Organize customers, only with names
